# Developer-journey
我的学习日志、进度追踪器和知识库
## 学习目标和计划

 **📅 第一阶段：Python进阶与Web开发破冰（第1-4周）**

**目标：从「写脚本」转向「做项目」，建立完整的Web开发概念**
### 第1周：Python工程化进阶
**工作日安排**
深入学习Python高级特性（装饰器、生成器、上下文管理器）
编写一个OOP风格的个人财务管理工具

**周末项目**
开发「智能文件整理器」（按扩展名/日期自动分类文件）
**产出**：GitHub仓库 'python-file-organizer'，包含完整文档

### 第2周：Web开发入门与FastAPI
**工作日安排**
学习HTTP协议、RESTful API设计
FastAPI框架入门，创建第一个API端点
使用Swagger UI测试API，学习API文档自动化
**周末项目**
构建「个人书单管理API」（支持书籍的增删改查）
**产出**：GitHub仓库 'book-management-api'，包含在线API文档

### 第3周：数据库集成
**工作日安排**
学习SQLAlchemy ORM，连接SQLite数据库
为书单API添加数据库持久化
数据库迁移和种子数据
**周末项目**
升级书单API，添加用户-书籍关系
**产出**：更新仓库，展示数据库设计和关系建模

### 第4周：前端界面开发
**工作日安排**
HTML/CSS基础，JavaScript DOM操作
学习Fetch API，前后端数据交互
前端调试技巧
**周末项目**
为书单管理系统开发完整的前端界面
**产出**：新仓库 'fullstack-book-manager'，展示全栈能力

**阶段成果**：拥有一个完整的前后端分离项目，理解Web开发全流程

3. **项目导向**：学习的每个新知识点必须立即应用到具体项目中
4. **社区参与**：关注技术社区（掘金、GitHub Trending），保持技术敏感度：配置生产环境变量和数据库
- **周末项目**：将2个主要项目部署到公网，确保可访问
- **产出**：所有项目README都有「在线演示」链接
